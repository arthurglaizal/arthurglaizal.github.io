Shards, DATIVE Design System

Brief

Realization and maintenance of the Design System of DATIVE, a SaaS start up.

This Design System is an adaptation of Vuetify which itself comes from Material from Google. The design system allowed a considerable saving of time in development and a homogeneous and optimized user experience.

The approach of the design system evolution was the following: Relying as much as possible on Vuetify components, updating the Dative graphic charter and creating/adopting components if a specific user need justifies it.

A beta version is available on the Storybook tool. It allows developers to test components in isolation and to identify differences between the desired design system and the production version:

https://dative-gpi.github.io/shards/
